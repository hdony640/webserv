Welcome to our Webserv

Webserv is a project developed at 42 that aims to create a basic HTTP server. It supports handling HTTP requests, processing them, and sending appropriate responses. In the world of web development, understanding the concepts of HTTP, status codes, and server-side processing is crucial. With Webserv, you can gain hands-on experience with server-side programming and deepen your understanding of web technologies. Feel free to explore the project's code on GitHub and check out the project's documentation for more details.
